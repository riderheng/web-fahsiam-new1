# Product Data Schema Reference\n\nThis document defines the complete data structure for fertilizer products in the Fah Siam website.\n\n## Product Object Structure\n\n```typescript\ntype Product = {\n  id: string;                    // Unique identifier (auto-generated from product name)\n  name: string;                  // Product name in Thai (required, min 3 chars)\n  price: number;                 // Current price in THB (required, must be > 0)\n  oldPrice?: number;             // Original price before discount (optional, must be > price)\n  badge?: \"ใหม่\" | \"ฮิต\" | \"ลดราคา\";  // Product badge (optional)\n  image: string;                 // Path to product image (required, must be .webp, .jpg, or .png)\n  category: string;              // Product category (required, currently only \"ปุ๋ย\")\n  benefits: string[];            // List of product benefits (required, non-empty array)\n  usages: ProductUsage[];        // Usage instructions by crop type (required, non-empty array)\n  updatedAt: string;             // Last update date in YYYY-MM-DD format (auto-generated)\n};\n```\n\n## ProductUsage Object Structure\n\n```typescript\ntype ProductUsage = {\n  groupName: string;             // Crop/plant group name (e.g., \"ทุเรียน\", \"ยางพารา\")\n  items: ProductUsageDetail[];   // Usage details for each specific crop\n};\n```\n\n## ProductUsageDetail Object Structure\n\n```typescript\ntype ProductUsageDetail = {\n  label: string;                 // Specific crop name (e.g., \"ทุเรียนอายุ 1-2 ปี\")\n  value: string;                 // Usage amount/instructions (e.g., \"100-300 กรัม/ต้น\")\n};\n```\n\n## Field Validation Rules\n\n### id (String)\n- **Format**: Auto-generated from product name\n- **Pattern**: lowercase with hyphens, ending with a suffix\n- **Example**: `organic-chemical-12-3-5-growth`\n- **Validation**: Must be unique across all products\n\n### name (String)\n- **Required**: Yes\n- **Min Length**: 3 characters\n- **Language**: Thai\n- **Example**: \"ปุ๋ยอินทรีย์เคมี 12 - 3 - 5\"\n\n### price (Number)\n- **Required**: Yes\n- **Type**: Integer or Float\n- **Constraint**: Must be > 0\n- **Currency**: Thai Baht (THB)\n- **Example**: 1290\n\n### oldPrice (Number)\n- **Required**: No (optional)\n- **Type**: Integer or Float\n- **Constraint**: Must be > price (if provided)\n- **Currency**: Thai Baht (THB)\n- **Example**: 1590\n- **Use Case**: Indicate discounted products\n\n### badge (String)\n- **Required**: No (optional)\n- **Valid Values**: \n  - `\"ใหม่\"` - New product\n  - `\"ฮิต\"` - Popular/trending product\n  - `\"ลดราคา\"` - Discounted product\n- **Example**: \"ลดราคา\"\n\n### image (String)\n- **Required**: Yes\n- **Path Format**: `/image/Fertilizer/{filename}.{ext}`\n- **Supported Formats**: `.webp` (preferred), `.jpg`, `.png`\n- **Location**: Must be in `public/image/Fertilizer/` directory\n- **Optimization**: Should be WebP format for best performance\n- **Example**: `/image/Fertilizer/1.webp`\n\n### category (String)\n- **Required**: Yes\n- **Valid Values**: Currently only `\"ปุ๋ย\"` (Fertilizer)\n- **Extensible**: Can be expanded for other product categories\n- **Example**: \"ปุ๋ย\"\n\n### benefits (Array of Strings)\n- **Required**: Yes\n- **Min Items**: 1\n- **Min Length per Item**: 3 characters\n- **Language**: Thai\n- **Purpose**: List product advantages and features\n- **Example**:\n  ```typescript\n  [\n    \"ช่วยบำรุงต้นและใบ\",\n    \"ช่วยให้รากแข็งแรงขึ้น\",\n    \"ช่วยให้พืชแตกยอดใหม่\"\n  ]\n  ```\n\n### usages (Array of ProductUsage)\n- **Required**: Yes\n- **Min Items**: 1\n- **Structure**: Each item must have `groupName` and `items` array\n- **Purpose**: Provide usage instructions for different crop types\n- **Example**:\n  ```typescript\n  [\n    {\n      groupName: \"ทุเรียน\",\n      items: [\n        { label: \"ทุเรียนอายุ 1-2 ปี\", value: \"100-300 กรัม/ต้น\" },\n        { label: \"ทุเรียนอายุ 3-4 ปี\", value: \"500 กรัม - 1 กก./ต้น\" }\n      ]\n    }\n  ]\n  ```\n\n### updatedAt (String)\n- **Required**: Yes\n- **Format**: `YYYY-MM-DD`\n- **Auto-generated**: Yes (current date when product is added/updated)\n- **Purpose**: Track product freshness for SEO\n- **Example**: \"2026-03-17\"\n\n## Common Product Types\n\n### Organic-Chemical Fertilizers\n- **Name Pattern**: \"ปุ๋ยอินทรีย์เคมี [NPK]-[NPK]-[NPK]\"\n- **Example ID**: `organic-chemical-12-3-5-growth`\n- **Common NPK Ratios**:\n  - 12-3-5: Growth/vegetative phase\n  - 3-6-15: Flowering/fruiting phase\n  - 6-3-3: All-purpose\n\n### Organic Powder Fertilizers\n- **Name Pattern**: \"ปุ๋ยอินทรีย์ผง OM [percentage]%\"\n- **Example ID**: `organic-powder-om-25-soil-booster`\n- **Common OM Percentages**: 20%, 25%\n\n### Chemical Fertilizers\n- **Name Pattern**: \"ปุ๋ยเคมี [NPK]-[NPK]-[NPK]\"\n- **Example ID**: `chemical-fertilizer-0-0-30-sweetness`\n- **Common NPK Ratios**: 0-0-30 (potassium for quality)\n\n### Organic Pellet Fertilizers\n- **Name Pattern**: \"ปุ๋ยอินทรีย์เม็ด OM [percentage]%\"\n- **Example ID**: `organic-pellet-om-20-slow-release`\n- **Common OM Percentages**: 20%, 25%\n\n## Crop Groups for Usages\n\nCommon crop groups used in the usages field:\n\n| Group Name | Crops Included | Example Usage |\n| :---------- | :------------- | :------------ |\n| ทุเรียน | Durian trees | 100-300 กรัม/ต้น |\n| ยางพารา | Rubber trees | 400-500 กรัม/ต้น/ปี |\n| ปาล์มน้ำมัน | Oil palm trees | 1.5-2 กก./ต้น/ปี |\n| ไม้ผล | Fruit trees | 1-3 กก./ต้น |\n| พืชผัก | Vegetables | 30-50 กก./ไร่ |\n| นาข้าว | Rice paddies | 25 กก./ไร่ |\n| ข้าวโพด | Corn | 25-50 กก./ไร่ |\n| อ้อย | Sugarcane | 50 กก./ไร่ |\n| มันสำปะหลัง | Cassava | 50 กก./ไร่ |\n| กาแฟ | Coffee | 200-400 กรัม/ต้น |\n| ยาสูบ | Tobacco | 50 กก./ไร่ |\n\n## Example Complete Product\n\n```typescript\n{\n  id: \"organic-chemical-12-3-5-growth\",\n  updatedAt: \"2026-03-12\",\n  name: \"ปุ๋ยอินทรีย์เคมี 12 - 3 - 5\",\n  price: 1290,\n  oldPrice: 1590,\n  badge: \"ลดราคา\",\n  image: \"/image/Fertilizer/1.webp\",\n  category: \"ปุ๋ย\",\n  benefits: [\n    \"ช่วยบำรุงต้นและใบ\",\n    \"ช่วยให้รากแข็งแรงขึ้น\",\n    \"ช่วยให้พืชแตกยอดใหม่\",\n    \"ช่วยให้ต้นพืชทนโรคและสภาพแวดล้อม\",\n    \"ช่วยขยายใบและเพิ่มการสังเคราะห์แสง\"\n  ],\n  usages: [\n    {\n      groupName: \"ทุเรียน\",\n      items: [\n        { label: \"ทุเรียนอายุ 1-2 ปี\", value: \"100-300 กรัม/ต้น\" },\n        { label: \"ทุเรียนอายุ 3-4 ปี\", value: \"500 กรัม - 1 กก./ต้น\" },\n        { label: label: \"ปีที่ 2-4\", value: \"600-700 กรัม/ต้น/ปี\" }\n      ]\n    }\n  ]\n}\n```\n\n## Data Entry Checklist\n\nWhen adding or updating a product, ensure:\n\n- [ ] Product name is in Thai and descriptive\n- [ ] Price is a positive number\n- [ ] oldPrice (if provided) is greater than price\n- [ ] Badge is one of the valid values\n- [ ] Image path exists and points to a WebP file\n- [ ] Category is \"ปุ๋ย\"\n- [ ] At least 3 benefits are provided\n- [ ] At least 1 usage group is provided\n- [ ] Each usage group has at least 1 item\n- [ ] All text is in Thai (except technical IDs)\n- [ ] No special characters in field values (except in labels/values)\n- [ ] updatedAt is auto-generated (don't manually set)\n\n## Notes\n\n- Product IDs are auto-generated and should not be manually modified\n- The `updatedAt` field is automatically set to the current date when products are added or updated\n- Images should be optimized to WebP format for best performance\n- All product names and descriptions should be in Thai\n- The system currently supports only one category (\"ปุ๋ย\"), but can be extended\n"
