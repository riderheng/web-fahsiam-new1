# SEO Guidelines for Fah Siam Website\n\nThis document provides SEO best practices for the Fah Siam fertilizer e-commerce website.\n\n## Overview\n\nSEO (Search Engine Optimization) is critical for an e-commerce website selling fertilizer products. The Fah Siam website targets farmers and agricultural businesses searching for quality fertilizer products in Thailand.\n\n## Target Keywords\n\n### Primary Keywords\n- ปุ๋ยอินทรีย์ (organic fertilizer)\n- ปุ๋ยเคมี (chemical fertilizer)\n- ปุ๋ยทุเรียน (durian fertilizer)\n- ปุ๋ยยางพารา (rubber tree fertilizer)\n- ปุ๋ยปาล์มน้ำมัน (oil palm fertilizer)\n- ปุ๋ยข้าว (rice fertilizer)\n- ปุ๋ยผัก (vegetable fertilizer)\n\n### Long-tail Keywords\n- ปุ๋ยอินทรีย์เคมี 12-3-5 สำหรับทุเรียน\n- ปุ๋ยเคมี 0-0-30 เพิ่มคุณภาพผลผลิต\n- ปุ๋ยอินทรีย์ผง OM 25% ปรับปรุงดิน\n- วิธีใช้ปุ๋ยสำหรับพืชไร่\n\n## On-Page SEO\n\n### Product Page Optimization\n\n#### Meta Tags\n```html\n<meta name=\"description\" content=\"ปุ๋ยอินทรีย์เคมี 12-3-5 ช่วยบำรุงต้นและใบ ให้รากแข็งแรง ราคา 1,290 บาท ใช้สำหรับทุเรียน ยางพารา ปาล์มน้ำมัน\" />\n<meta name=\"keywords\" content=\"ปุ๋ยอินทรีย์เคมี, ปุ๋ย 12-3-5, ปุ๋ยทุเรียน, ปุ๋ยอินทรีย์\" />\n<meta property=\"og:title\" content=\"ปุ๋ยอินทรีย์เคมี 12 - 3 - 5 | ฟ้าสยาม\" />\n<meta property=\"og:description\" content=\"ปุ๋ยอินทรีย์เคมี 12-3-5 ช่วยบำรุงต้นและใบ ให้รากแข็งแรง ราคา 1,290 บาท\" />\n<meta property=\"og:image\" content=\"/image/Fertilizer/1.webp\" />\n```\n\n#### Page Title\n- **Format**: \"[Product Name] | ฟ้าสยาม\"\n- **Length**: 50-60 characters\n- **Example**: \"ปุ๋ยอินทรีย์เคมี 12 - 3 - 5 | ฟ้าสยาม\"\n- **Best Practice**: Include main keyword and brand name\n\n#### Meta Description\n- **Length**: 150-160 characters\n- **Content**: Product name, key benefits, price\n- **Example**: \"ปุ๋ยอินทรีย์เคมี 12-3-5 ช่วยบำรุงต้นและใบ ให้รากแข็งแรง ราคา 1,290 บาท ใช้สำหรับทุเรียน ยางพารา ปาล์มน้ำมัน\"\n\n#### H1 Tag\n- **Should appear once**: Use product name as H1\n- **Example**: \"ปุ๋ยอินทรีย์เคมี 12 - 3 - 5\"\n\n#### URL Structure\n- **Format**: `/products/{product-id}`\n- **Example**: `/products/organic-chemical-12-3-5-growth`\n- **Best Practice**: Use descriptive, hyphenated IDs\n\n### Content Optimization\n\n#### Product Benefits Section\n- **Structure**: Use H2 for \"ประโยชน์ของผลิตภัณฑ์\"\n- **Format**: Bullet list with clear benefits\n- **Include**: Thai language, specific outcomes\n- **Example**:\n  - ช่วยบำรุงต้นและใบ\n  - ช่วยให้รากแข็งแรงขึ้น\n  - ช่วยให้พืชแตกยอดใหม่\n\n#### Usage Instructions Section\n- **Structure**: Use H2 for \"วิธีใช้งาน\"\n- **Format**: Organized by crop type (H3 headers)\n- **Content**: Specific dosage for each crop and age\n- **Example**:\n  ```\n  ### ทุเรียน\n  - ทุเรียนอายุ 1-2 ปี: 100-300 กรัม/ต้น\n  - ทุเรียนอายุ 3-4 ปี: 500 กรัม - 1 กก./ต้น\n  ```\n\n#### Image Optimization\n- **Format**: WebP (preferred), JPG, PNG\n- **Size**: Compressed to <200KB per image\n- **Alt Text**: Descriptive, include product name and benefits\n- **Example Alt Text**: \"ปุ๋ยอินทรีย์เคมี 12-3-5 สำหรับบำรุงต้นและใบ\"\n\n## Technical SEO\n\n### Sitemap\n- **File**: `app/sitemap.ts`\n- **Update Frequency**: Automatic when products are added/updated\n- **Include**: All product pages, main pages, category pages\n- **Priority**: Product pages (0.8), main pages (1.0), category pages (0.9)\n- **Change Frequency**: \n  - Home: daily\n  - Products: daily\n  - Product details: weekly\n  - News: weekly\n  - Contact: monthly\n\n### Robots.txt\n```\nUser-agent: *\nAllow: /\nDisallow: /admin\nDisallow: /api\nSitemap: https://puripong1st.github.io/web-fahsiam/sitemap.xml\n```\n\n### Structured Data (Schema.org)\n\n#### Product Schema\n```json\n{\n  \"@context\": \"https://schema.org/\",\n  \"@type\": \"Product\",\n  \"name\": \"ปุ๋ยอินทรีย์เคมี 12 - 3 - 5\",\n  \"description\": \"ปุ๋ยอินทรีย์เคมี 12-3-5 ช่วยบำรุงต้นและใบ ให้รากแข็งแรง\",\n  \"image\": \"/image/Fertilizer/1.webp\",\n  \"brand\": {\n    \"@type\": \"Brand\",\n    \"name\": \"ฟ้าสยาม\"\n  },\n  \"offers\": {\n    \"@type\": \"Offer\",\n    \"price\": \"1290\",\n    \"priceCurrency\": \"THB\",\n    \"availability\": \"https://schema.org/InStock\"\n  }\n}\n```\n\n### Mobile Optimization\n- **Responsive Design**: All pages must be mobile-friendly\n- **Page Speed**: Target <3 seconds load time\n- **Viewport Meta Tag**: `<meta name=\"viewport\" content=\"width=device-width, initial-scale=1\">`\n\n### Core Web Vitals\n- **Largest Contentful Paint (LCP)**: <2.5 seconds\n- **First Input Delay (FID)**: <100 milliseconds\n- **Cumulative Layout Shift (CLS)**: <0.1\n\n## Off-Page SEO\n\n### Backlink Strategy\n- **Target**: Agricultural websites, farming forums, product review sites\n- **Anchor Text**: Include product names and keywords\n- **Quality over Quantity**: Focus on relevant, high-authority sites\n\n### Social Media Integration\n- **Platforms**: Facebook, Instagram, Line\n- **Content**: Product updates, farming tips, customer testimonials\n- **Frequency**: At least 2-3 posts per week\n\n## Content Strategy\n\n### Blog/News Section\n- **Topic Ideas**:\n  - \"วิธีเลือกปุ๋ยที่เหมาะสมสำหรับพืชของคุณ\"\n  - \"ประโยชน์ของปุ๋ยอินทรีย์ต่อคุณภาพดิน\"\n  - \"ปัญหาการใช้ปุ๋ยและวิธีแก้ไข\"\n  - \"ฤดูกาลที่เหมาะสมสำหรับการใช้ปุ๋ย\"\n\n- **Frequency**: At least 1 post per month\n- **Length**: 800-1500 words per article\n- **Keywords**: Include target keywords naturally\n\n### FAQ Section\n- **Common Questions**:\n  - \"ปุ๋ยอินทรีย์กับปุ๋ยเคมีต่างกันอย่างไร?\"\n  - \"ควรใช้ปุ๋ยเท่าไหร่ต่อไร่?\"\n  - \"เมื่อไหร่ที่ควรใช้ปุ๋ย?\"\n  - \"ปุ๋ยใดดีที่สุดสำหรับทุเรียน?\"\n\n## Local SEO\n\n### Google Business Profile\n- **Business Name**: ฟ้าสยาม\n- **Category**: Fertilizer Supplier\n- **Location**: Thailand-wide (if applicable)\n- **Phone**: Include contact number\n- **Website**: Link to website\n\n### Local Keywords\n- Include city/region names in content\n- Example: \"ปุ๋ยอินทรีย์เคมี จังหวัรสุโคศ\" (if location-specific)\n\n## Monitoring & Analytics\n\n### Google Search Console\n- **Monitor**: Impressions, clicks, average position\n- **Fix**: Crawl errors, mobile usability issues\n- **Submit**: Sitemap, URL inspection\n\n### Google Analytics\n- **Track**: Organic traffic, user behavior, conversions\n- **Goals**: Product page views, contact form submissions\n- **Reports**: Traffic sources, user demographics\n\n### SEO Metrics to Monitor\n- **Organic Traffic**: Monthly trend\n- **Keyword Rankings**: Top 20 keywords\n- **Bounce Rate**: Target <50% for product pages\n- **Average Session Duration**: Target >2 minutes\n- **Conversion Rate**: Track product inquiries/purchases\n\n## Maintenance Checklist\n\n### Monthly\n- [ ] Check Google Search Console for errors\n- [ ] Review top-performing keywords\n- [ ] Update product prices if changed\n- [ ] Check for broken links\n\n### Quarterly\n- [ ] Audit product descriptions for keyword optimization\n- [ ] Review and update meta descriptions\n- [ ] Check mobile performance\n- [ ] Analyze competitor keywords\n\n### Annually\n- [ ] Comprehensive SEO audit\n- [ ] Update content strategy\n- [ ] Review and optimize internal linking\n- [ ] Assess backlink profile\n\n## Common SEO Mistakes to Avoid\n\n1. **Duplicate Content**: Each product page should have unique descriptions\n2. **Missing Alt Text**: Always include descriptive alt text for images\n3. **Slow Page Speed**: Optimize images and minimize CSS/JavaScript\n4. **Poor Mobile Experience**: Test on various devices\n5. **Keyword Stuffing**: Use keywords naturally, not excessively\n6. **Broken Links**: Regularly check for 404 errors\n7. **Missing Meta Tags**: Always include title and description\n8. **Outdated Content**: Keep product information current\n\n## Tools & Resources\n\n### Free Tools\n- Google Search Console: https://search.google.com/search-console\n- Google Analytics: https://analytics.google.com/\n- Google PageSpeed Insights: https://pagespeed.web.dev/\n- Ubersuggest: https://ubersuggest.com/\n\n###
