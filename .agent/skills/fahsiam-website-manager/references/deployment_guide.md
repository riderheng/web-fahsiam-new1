# Deployment Guide for Fah Siam Website\n\nThis guide provides step-by-step instructions for deploying the Fah Siam website to production.\n\n## Prerequisites\n\nBefore deploying, ensure you have:\n\n1. **GitHub Account**: With access to the `puripong1st/web-fahsiam` repository\n2. **Git Installed**: On your local machine or deployment environment\n3. **Node.js & npm**: Version 18+ installed\n4. **GitHub CLI**: For automated deployments\n5. **GitHub Pages Enabled**: Repository settings configured for GitHub Pages\n\n## Deployment Environments\n\n### Development\n- **URL**: `http://localhost:3000`\n- **Branch**: `develop`\n- **Build**: `npm run dev`\n- **Purpose**: Testing and development\n\n### Staging\n- **URL**: `https://puripong1st.github.io/web-fahsiam-staging`\n- **Branch**: `staging`\n- **Build**: `npm run build`\n- **Purpose**: Pre-production testing\n\n### Production\n- **URL**: `https://puripong1st.github.io/web-fahsiam`\n- **Branch**: `main`\n- **Build**: `npm run build`\n- **Purpose**: Live website\n\n## Pre-Deployment Checklist\n\nBefore deploying to production:\n\n- [ ] All changes committed to Git\n- [ ] Product data validated (run `validate_products.py`)\n- [ ] Images optimized (run `optimize_images.py`)\n- [ ] Sitemap updated (run `update_sitemap.py`)\n- [ ] No broken links (test locally)\n- [ ] Mobile responsiveness verified\n- [ ] Page speed acceptable (<3 seconds)\n- [ ] Meta tags and descriptions reviewed\n- [ ] All tests passing (if applicable)\n\n## Deployment Process\n\n### Step 1: Prepare Changes\n\n```bash\n# Navigate to repository\ncd /home/ubuntu/web-fahsiam\n\n# Ensure you're on the correct branch\ngit checkout main\n\n# Pull latest changes\ngit pull origin main\n```\n\n### Step 2: Validate All Data\n\n```bash\n# Validate products\npython /home/ubuntu/skills/fahsiam-website-manager/scripts/validate_products.py /home/ubuntu/web-fahsiam\n\n# Update sitemap\npython /home/ubuntu/skills/fahsiam-website-manager/scripts/update_sitemap.py /home/ubuntu/web-fahsiam\n```\n\n### Step 3: Build the Project\n\n```bash\n# Install dependencies (if needed)\nnpm install\n\n# Build the Next.js project\nnpm run build\n\n# Verify build succeeded\nif [ $? -eq 0 ]; then\n  echo \"✅ Build successful\"\nelse\n  echo \"❌ Build failed\"\n  exit 1\nfi\n```\n\n### Step 4: Test Locally\n\n```bash\n# Start the production server\nnpm run start\n\n# Test at http://localhost:3000\n# Verify:\n# - Home page loads correctly\n# - Product pages display properly\n# - Images load without errors\n# - Navigation works\n# - Mobile responsive\n\n# Stop the server (Ctrl+C)\n```\n\n### Step 5: Commit and Push\n\n```bash\n# Check git status\ngit status\n\n# Stage all changes\ngit add .\n\n# Commit with descriptive message\ngit commit -m \"Deploy: Update products and sitemap - $(date +%Y-%m-%d)\"\n\n# Push to main branch\ngit push origin main\n```\n\n### Step 6: Deploy to GitHub Pages\n\nGitHub Pages automatically deploys when you push to the `main` branch. The deployment typically takes 1-5 minutes.\n\n```bash\n# Monitor deployment status\ngh run list --repo puripong1st/web-fahsiam\n\n# View deployment logs\ngh run view <run_id> --repo puripong1st/web-fahsiam\n```\n\n### Step 7: Verify Deployment\n\n```bash\n# Check if site is live\ncurl -I https://puripong1st.github.io/web-fahsiam\n\n# Verify sitemap is accessible\ncurl -I https://puripong1st.github.io/web-fahsiam/sitemap.xml\n\n# Test a product page\ncurl -I https://puripong1st.github.io/web-fahsiam/products/organic-chemical-12-3-5-growth\n```\n\n## Automated Deployment Script\n\nYou can use the provided `deploy_website.sh` script for automated deployment:\n\n```bash\nbash /home/ubuntu/skills/fahsiam-website-manager/scripts/deploy_website.sh /home/ubuntu/web-fahsiam\n```\n\nThis script performs all steps automatically:\n1. Validates product data\n2. Updates sitemap\n3. Builds the project\n4. Commits changes\n5. Pushes to GitHub\n6. Monitors deployment\n\n## Troubleshooting\n\n### Build Fails\n\n**Error**: `npm ERR! code ENOENT`\n\n**Solution**:\n```bash\n# Clear node_modules and reinstall\nrm -rf node_modules package-lock.json\nnpm install\nnpm run build\n```\n\n**Error**: `Module not found`\n\n**Solution**:\n```bash\n# Check for missing dependencies\nnpm install\n\n# Verify imports in affected files\n# Check for typos in import paths\n```\n\n### Deployment Doesn't Appear\n\n**Issue**: Changes don't appear on live site\n\n**Solutions**:\n1. Check GitHub Actions workflow status\n2. Clear browser cache (Ctrl+Shift+Delete)\n3. Wait 5-10 minutes for GitHub Pages to update\n4. Verify changes are on the `main` branch\n5. Check GitHub Pages settings in repository\n\n### Page Speed Issues\n\n**Issue**: Site loads slowly\n\n**Solutions**:\n1. Optimize images using `optimize_images.py`\n2. Minimize CSS/JavaScript\n3. Enable gzip compression\n4. Use CDN for static assets\n5. Check for large dependencies\n\n### Broken Links After Deployment\n\n**Issue**: Links work locally but not on production\n\n**Solutions**:\n1. Verify base path in `next.config.ts` is correct\n2. Check for absolute vs. relative paths\n3. Test with production URL\n4. Check GitHub Pages settings\n\n## Rollback Procedure\n\nIf deployment causes issues, you can rollback to a previous version:\n\n```bash\n# View commit history\ngit log --oneline -10\n\n# Revert to previous commit\ngit revert <commit_hash>\n\n# Or reset to previous commit (use with caution)\ngit reset --hard <commit_hash>\n\n# Push changes\ngit push origin main\n```\n\n## Performance Optimization\n\n### Image Optimization\n\n```bash\n# Optimize all product images\npython /home/ubuntu/skills/fahsiam-website-manager/scripts/optimize_images.py /home/ubuntu/web-fahsiam /path/to/images/*\n```\n\n### Build Optimization\n\n```bash\n# Enable Next.js optimizations\n# In next.config.ts:\n{\n  images: {\n    formats: ['image/avif', 'image/webp'],\n    deviceSizes: [640, 750, 828, 1080, 1200, 1920, 2048, 3840],\n    imageSizes: [16, 32, 48, 64, 96, 128, 256, 384],\n  },\n  compress: true,\n  swcMinify: true,\n}\n```\n\n## Monitoring After Deployment\n\n### Google Search Console\n1.
