# SEO Content Writing Guidelines & Standard Prompts

แนวทางการเขียนเนื้อหาให้ติดอันดับและชุดคำสั่งมาตรฐานสำหรับ AI ในการทำ SEO

## 1. แนวทางการเขียนเนื้อหา (Content Guidelines)

*   **E-E-A-T:** แสดงความเชี่ยวชาญ (Expertise), ประสบการณ์ (Experience), ความน่าเชื่อถือ (Authoritativeness), และความไว้วางใจ (Trustworthiness)
*   **Topic Clusters:** เขียนเนื้อหาที่ครอบคลุมหัวข้อหลักและหัวข้อย่อยที่เกี่ยวข้อง เพื่อสร้าง Topic Authority
*   **Semantic Richness:** ใช้คำศัพท์ที่เกี่ยวข้องกับอุตสาหกรรมนั้นๆ เพื่อช่วยให้ Search Engine เข้าใจบริบท
*   **Readability:** ใช้ประโยคที่กระชับ แบ่งย่อหน้าสม่ำเสมอ และใช้หัวข้อ (Subheadings) เพื่อให้อ่านง่ายทั้งคนและ Bot

## 2. ชุดคำสั่งมาตรฐาน (Standard Prompts)

### 2.1 สำหรับการวิจัย Keyword
> "ช่วยวิเคราะห์ Keyword สำหรับเว็บไซต์ [ระบุประเภทธุรกิจ] โดยเน้นไปที่กลุ่มเป้าหมาย [ระบุกลุ่มเป้าหมาย] โปรดระบุ Keyword หลัก, Long-tail Keywords ที่มีการแข่งขันต่ำแต่มีคุณภาพ และคำศัพท์ที่เกี่ยวข้อง (LSI) พร้อมอธิบายเหตุผลที่เลือก"

### 2.2 สำหรับการเขียน Meta Description
> "ช่วยเขียน Meta Description สำหรับหน้า [ระบุชื่อหน้า] ของเว็บไซต์ [ระบุชื่อเว็บ] โดยให้มีความยาวระหว่าง 150-160 ตัวอักษร มี Keyword [ระบุ Keyword] และมี Call to Action ที่ดึงดูดใจผู้ใช้"

### 2.3 สำหรับการปรับปรุงเนื้อหา (On-page)
> "ช่วยตรวจสอบเนื้อหาในหน้านี้ [แนบเนื้อหาหรือ URL] และแนะนำการปรับปรุงตามหลัก SEO โดยเน้นการวาง Heading Tags (H1-H3), การกระจาย Keyword อย่างเป็นธรรมชาติ และการเพิ่มความน่าเชื่อถือของเนื้อหา"

### 2.4 สำหรับการสร้าง Schema.org
> "ช่วยสร้าง JSON-LD สำหรับ [ระบุประเภท เช่น Product/FAQ] โดยใช้ข้อมูลดังนี้ [แนบข้อมูล] และตรวจสอบให้แน่ใจว่าถูกต้องตามมาตรฐานของ Schema.org และ Google Rich Results"
