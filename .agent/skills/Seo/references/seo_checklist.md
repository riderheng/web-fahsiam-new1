# SEO & Metadata Implementation Checklist

รายการตรวจสอบความสมบูรณ์แบบสำหรับการทำ SEO และ Metadata เพื่อให้มั่นใจว่าไม่มีจุดใดขาดตกบกพร่อง

## 1. On-Page SEO Checklist
- [ ] **Title Tag:** มี Keyword หลักอยู่ด้านหน้า, ความยาว 50-60 ตัวอักษร, ดึงดูดการคลิก
- [ ] **Meta Description:** มี Keyword หลักและรอง, ความยาว 150-160 ตัวอักษร, มี Call to Action (CTA)
- [ ] **Heading Tags (H1-H6):** มี H1 เพียงหนึ่งเดียวต่อหน้า, ใช้ H2-H3 แบ่งหัวข้อย่อยอย่างเป็นระบบ
- [ ] **URL Structure:** สั้น, สื่อความหมาย, ใช้ยัติภังค์ (-) แทนช่องว่าง, มี Keyword
- [ ] **Image Optimization:** มี Alt Text ทุกรูป, ใช้ไฟล์ WebP/AVIF, ขนาดไฟล์ไม่เกิน 100KB
- [ ] **Internal Linking:** เชื่อมโยงไปยังหน้าสำคัญภายในเว็บด้วย Anchor Text ที่สื่อความหมาย
- [ ] **External Linking:** ลิงก์ไปยังแหล่งอ้างอิงที่มีความน่าเชื่อถือ (Authority Sites)

## 2. Technical SEO Checklist
- [ ] **Canonical URL:** กำหนด URL หลักเพื่อป้องกัน Duplicate Content
- [ ] **Robots.txt:** ตั้งค่าการอนุญาต/ไม่อนุญาตให้ Bot เข้าถึงหน้าต่างๆ อย่างถูกต้อง
- [ ] **XML Sitemap:** มีไฟล์ Sitemap ที่อัปเดตล่าสุดและส่งให้ Google Search Console
- [ ] **Mobile-Friendliness:** ตรวจสอบการแสดงผลบนมือถือให้สมบูรณ์ (Responsive Design)
- [ ] **Page Speed:** คะแนน Google PageSpeed Insights (Core Web Vitals) อยู่ในเกณฑ์สีเขียว
- [ ] **HTTPS:** เว็บไซต์ต้องใช้งานผ่านโปรโตคอลความปลอดภัย SSL/TLS
- [ ] **Structured Data (Schema.org):** ติดตั้ง JSON-LD สำหรับ Organization, Product, Article, FAQ ตามความเหมาะสม

## 3. Semantic & Content SEO Checklist
- [ ] **User Intent:** เนื้อหาตอบโจทย์สิ่งที่ผู้ใช้ค้นหาจริง (Informational, Navigational, Transactional)
- [ ] **Keyword Density & LSI:** ใช้คำที่เกี่ยวข้องทางความหมาย (Latent Semantic Indexing) อย่างเป็นธรรมชาติ
- [ ] **Readability:** ภาษาอ่านง่าย, แบ่งย่อหน้าชัดเจน, ใช้ Bullet Points/Tables ช่วยในการอ่าน
- [ ] **Freshness:** เนื้อหามีความทันสมัยและอัปเดตข้อมูลล่าสุดเสมอ

## 4. Social & Metadata Checklist
- [ ] **Open Graph (OG):** กำหนด og:title, og:description, og:image สำหรับ Facebook/Line
- [ ] **Twitter Card:** กำหนด twitter:card, twitter:title, twitter:image สำหรับ X (Twitter)
- [ ] **Favicon & App Icons:** มีไอคอนครบทุกขนาดสำหรับเบราว์เซอร์และอุปกรณ์ต่างๆ
